把 `frontend/src/main.js` 里 YouTube 相关部分按下面修一下。

1. 确保 `state.youtube` 里真的有这些字段：

      youtube: {
        urlInput: '',
        channelName: 'manual_channel',
        items: [],
        authorFilter: '',
        startTime: '',
        endTime: '',
        statusFilter: '',
        taskId: '',
        progress: null,
      },

2. 把 `loadYoutube` 改成这个更稳的版本，不直接对 undefined 调 `trim()`：

    const loadYoutube = async () => safeRun(async () => {
      const params = { limit: 100 };
      const authorFilter = state.youtube?.authorFilter || '';
      const startTime = state.youtube?.startTime || '';
      const endTime = state.youtube?.endTime || '';
      const statusFilter = state.youtube?.statusFilter || '';

      if (authorFilter.trim()) params.author_name = authorFilter.trim();
      if (startTime) params.start_time = new Date(startTime).toISOString();
      if (endTime) params.end_time = new Date(endTime).toISOString();
      if (statusFilter) params.analysis_status = statusFilter;

      const { data } = await api.youtubeList(params);
      state.youtube.items = data;
      setStatus(`YouTube 条目 ${data.length} 条`);
    });

3. 如果你还加了 `importYoutube`、`pollYoutubeTask`、`analyzeAllYoutube`，也建议统一这样取值：

const authorFilter = state.youtube?.authorFilter || '';

不要直接写：

state.youtube.authorFilter.trim()

4. 如果页面模板里用了这些字段，也确保对应输入框绑定的字段名和 `state.youtube` 里的字段完全一致：

state.youtube.authorFilter
state.youtube.startTime
state.youtube.endTime
state.youtube.statusFilter
